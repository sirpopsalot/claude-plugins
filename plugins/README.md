# Plugins

Personal [Claude Code](https://docs.anthropic.com/en/docs/claude-code) plugins.

## Available Plugins

| Plugin | Description | Install |
|--------|-------------|---------|
| [freelance-finance-tracker](./freelance-finance-tracker/) | Skills for freelance income/expense tracking, tax estimates, and Gmail capture | `/plugin install freelance-finance-tracker@sirpopsalot` |
| [triathlon-coach](./triathlon-coach/) | Triathlon training plans and workout guidance | `/plugin install triathlon-coach@sirpopsalot` |
| [daily-ai-briefing](./daily-ai-briefing/) | Scan AI news sources and generate a structured daily briefing | `/plugin install daily-ai-briefing@sirpopsalot` |
